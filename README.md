
# MindSpace

MindSpace is a simple and aesthetic wellness web application built using React.js. It provides a calm and relaxing environment where users can explore motivational quotes, mood-based quotes, and wellness tips.

## Features

* Daily Motivational Quotes
* Mood-Based Quotes
* Positive Thinking Section
* Aesthetic Green Theme
* Smooth Navigation with React Router
* Responsive User Interface

## Technologies Used

* React.js
* React Router DOM
* JavaScript
* CSS
* useState Hook
* Components and Props


## Purpose

This project was developed as a Semester Project for **Advanced Web Engineering**.
